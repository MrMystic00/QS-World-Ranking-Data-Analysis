# 🎓 QS World University Rankings Analysis

## 📌 Project Overview

This project explores the factors influencing university rankings using the **QS World University Rankings 2024** dataset obtained from Kaggle. The analysis focuses on understanding how institutional characteristics and performance indicators relate to global university rankings through statistical analysis and regression techniques.

The project aims to:

* Understand the impact of university characteristics on rankings
* Explore relationships between academic reputation and performance metrics
* Investigate how geographical location influences international faculty scores

---

# 📂 Dataset

The dataset used in this project was extracted from **Kaggle**.

* **Dataset:** QS World University Rankings 2024
* **Source:** Kaggle

---

# ❓ Research Questions

The project investigates the following research questions:

1. Among the categorical variables:

   * Research intensity
   * University age
   * Focus
   * Institution size
   * Status

   Which of these factors significantly affect university rankings?

2. Does the continent where a university is located influence its international faculty score?

3. What relationships exist between academic reputation scores and other university performance measures?

# 🔍 Exploratory Data Analysis & Data Cleaning

Performed exploratory data analysis (EDA) to better understand the dataset and improve data quality.

Key processes include:

* Identifying and removing outliers
* Handling variables with insufficient category representation
* Visualizing distributions and relationships using:

  * Bar plots
  * Box plots
  * Histograms

---

# 🔄 Data Transformation

To improve statistical analysis and model assumptions:

* Applied **log transformation** to the **Overall Raw Score**
* Reduced right-skewness in the distribution
* Improved normality for regression analysis

---

# 📈 Statistical Analysis & Modeling

Used statistical testing and linear regression techniques to analyze relationships between variables.

Methods include:

* Correlation analysis
* Hypothesis testing
